Representation Learning for Non-Melanoma Skin Cancer

S.M.Thomas | 25th August 2022

Australian e-Health Research Centre, CSIRO

This article explores ideas presented in the paper Representation Learning for Non-Melanoma Skin Cancer using Latent Autoencoders. The accompanying code is available in the Github repository.


Generative models tend to have good representaions for a distribution, without explicitly being asked to model the input data. For examle, GANs implicity match the target distirbution by trying to generate plausible counterfeit images. However, autoecoder methods attempt to explicity model the target distribution, learning a latent representation constrained on reconstructing real images. Indeed, it is real images which are our best examples of what we want, and so being able to reconstruct them (at least initially) is a reasonable starting point. The problem is that Autoecoders don't tend to place images in a strucutured latent space. That is, not every image encoding within the representation corresponds to a real (our plausiably real) image. GANs on the other hand, do exactly that. Every point within the input domain should correspond to a realistic image.

Adversarial Latent autoencoders are a model design which attempts to get the benefits of a strucutured latents space, as well as the ablility to encode real images.
